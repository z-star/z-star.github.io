Java GC机制主要完成3件事：
  1. 确定哪些内存需要回收
  2. 确定什么时候需要执行GC
  3. 如何执行GC。
学习Java GC机制
  1. 内存是如何分配的
  2. 如何保证内存不被错误回收（即：哪些内存需要回收）
  3. 在什么情况下执行GC以及执行GC的方式
  4. 如何监控和优化GC机制。
